# Superstore Sales Analytics

## Overview
End to end sales analytics project analyzing 51K+ 
global retail records using MySQL and Power BI.

## Tools Used
- MySQL Workbench
- Power BI Desktop
- Dataset: Superstore Global Sales

## Dashboard Pages
- Summary Dashboard
- Executive Summary
- Product Analysis
- Regional Performance
- Customer Intelligence

## Key Insights
- APAC region generates highest profit at 0.44M
- Tables is the only loss making sub category
- High discounting directly reduces profit
- Consumer segment drives maximum sales
- Standard Class is most used shipping mode
